## 📌 今日题目：
和为 K 的子数组

### 📝 题目描述

给定一个整数数组 `nums` 和一个整数 `k`，请你统计并返回该数组中和为 `k` 的连续子数组的个数。

---

### 示例  

**示例 1：**  
输入：`nums = [1,1,1]`, `k = 2`  
输出：`2`  
解释：数组中有两个连续子数组的和为 2：[1,1]（下标 0~1），[1,1]（下标 1~2）。

**示例 2：**  
输入：`nums = [1,2,3]`, `k = 3`  
输出：`2`  
解释：[1,2] 和 [3]。

---

## 思路分析

本题的核心是**前缀和 + 哈希表**。

- 用一个变量 `pre_sum` 记录前缀和。
- 用哈希表 `count` 记录每个前缀和出现的次数。
- 遍历数组，对于每个位置，判断 `pre_sum - k` 是否在哈希表中，若存在则说明有以当前位置结尾、和为 k 的子数组。
- 每次更新哈希表中当前前缀和的出现次数。
- 时间复杂度 O(n)，空间复杂度 O(n)。

---

## ✅ Python 解法
```python
def subarray_sum(nums, k):
    from collections import defaultdict
    count = defaultdict(int)
    count[0] = 1
    pre_sum = 0
    res = 0
    for num in nums:
        pre_sum += num
        res += count[pre_sum - k]
        count[pre_sum] += 1
    return res
```

## ✅ Java 解法
```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        Map<Integer, Integer> count = new HashMap<>();
        count.put(0, 1);
        int preSum = 0, res = 0;
        for (int num : nums) {
            preSum += num;
            res += count.getOrDefault(preSum - k, 0);
            count.put(preSum, count.getOrDefault(preSum, 0) + 1);
        }
        return res;
    }
}
```
