## 📌 今日题目：
找到字符串中所有字母异位词

### 📝 题目描述

给定两个字符串 `s` 和 `p`，在 `s` 中找到所有 `p` 的字母异位词的起始索引，并以列表形式返回。

字母异位词指由相同字母组成、顺序可以不同的字符串。

---

### 示例  

**示例 1：**  
输入：`s = "cbaebabacd"`, `p = "abc"`  
输出：`[0, 6]`  
解释：起始索引 0 的子串是 "cba"，起始索引 6 的子串是 "bac"，它们都是 "abc" 的字母异位词。

**示例 2：**  
输入：`s = "abab"`, `p = "ab"`  
输出：`[0, 1, 2]`

---

## 思路分析

本题的核心是**滑动窗口 + 计数数组**。

- 用两个长度为 26 的数组分别统计当前窗口和目标字符串 `p` 的字母出现频率。
- 先初始化第一个窗口，然后每次滑动窗口时，更新计数数组。
- 每次比较两个计数数组是否相等，相等则说明当前窗口是异位词。
- 时间复杂度 O(n)，空间复杂度 O(1)。

---

## ✅ Python 解法
```python
def find_anagrams(s, p):
    from collections import Counter
    res = []
    p_len, s_len = len(p), len(s)
    if s_len < p_len:
        return res
    p_count = [0] * 26
    s_count = [0] * 26
    for i in range(p_len):
        p_count[ord(p[i]) - ord('a')] += 1
        s_count[ord(s[i]) - ord('a')] += 1
    if s_count == p_count:
        res.append(0)
    for i in range(p_len, s_len):
        s_count[ord(s[i - p_len]) - ord('a')] -= 1
        s_count[ord(s[i]) - ord('a')] += 1
        if s_count == p_count:
            res.append(i - p_len + 1)
    return res
```

## ✅ Java 解法
```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        List<Integer> res = new ArrayList<>();
        int sLen = s.length(), pLen = p.length();
        if (sLen < pLen) return res;
        int[] scount = new int[26];
        int[] pcount = new int[26];
        for (int i = 0; i < pLen; i++) {
            scount[s.charAt(i) - 'a']++;
            pcount[p.charAt(i) - 'a']++;
        }
        if (Arrays.equals(scount, pcount)) res.add(0);
        for (int i = pLen; i < sLen; i++) {
            scount[s.charAt(i - pLen) - 'a']--;
            scount[s.charAt(i) - 'a']++;
            if (Arrays.equals(scount, pcount)) res.add(i - pLen + 1);
        }
        return res;
    }
}
```
