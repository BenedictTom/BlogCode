# LeetCode Hot 100 算法题解

## 两数之和
**思路：** 
这道题的关键是快速查找。暴力解法需要两层循环，时间复杂度O(n²)。我们可以用哈希表优化到O(n)。

核心思想是：对于每个数字nums[i]，我们需要找另一个数字target - nums[i]。如果这个数字之前出现过，我们就找到了答案。哈希表可以在O(1)时间内完成查找。

**步骤：**
1. 创建一个哈希表，用于存储数字和它的索引的映射关系
2. 遍历数组，对于每个数字nums[i]：
   - 计算需要的另一个数字：need = target - nums[i]
   - 检查need是否在哈希表中
   - 如果在，返回当前索引i和need对应的索引
   - 如果不在，将当前数字和索引存入哈希表
3. 如果遍历完都没找到，返回null

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        // 用哈希表存储：数字 -> 索引
        HashMap<Integer, Integer> map = new HashMap<Integer,Integer>();
        for (int i = 0; i< nums.length; i++){
            // 检查目标值减去当前数字是否在哈希表中
            if (map.containsKey(target-nums[i])){
                return new int[]{i, map.get(target-nums[i])};
            }
            // 将当前数字和索引存入哈希表
            map.put(nums[i],i);
        }
        return null;
    }
}
```

## 字母异位词分组
**思路：**
字母异位词是指字母相同但排列不同的字符串，比如"eat"和"tea"。关键观察是：如果两个字符串是异位词，那么它们排序后的结果一定相同。

我们可以利用这个性质，将排序后的字符串作为分组的key，所有排序后相同的字符串就是一组异位词。

**步骤：**
1. 创建一个哈希表，key是排序后的字符串，value是异位词列表
2. 遍历字符串数组：
   - 将当前字符串转为字符数组并排序
   - 将排序后的字符数组转为字符串作为key
   - 从哈希表中获取对应的列表（如果不存在则创建新列表）
   - 将原字符串加入列表
   - 将列表存入哈希表
3. 返回哈希表中所有列表的集合

```java
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        // key: 排序后的字符串, value: 异位词列表
        Map<String, List<String>> map = new HashMap();
        for (int i=0; i< strs.length; i++){
            // 将字符串转为字符数组并排序
            char[] charArray = strs[i].toCharArray();
            Arrays.sort(charArray);
            String key = new String(charArray);
            // 获取或创建对应的列表
            List<String> list = map.getOrDefault(key, new ArrayList<>());
            list.add(strs[i]);
            map.put(key,list);
        }
        // 返回所有列表
        return new ArrayList<List<String>>(map.values());
    }
}
```

## 最长连续序列
**思路：**
这道题要求找出数字连续的最长序列的长度。关键是要在O(n)时间内完成。

核心思想是：只从序列的起点开始计算。如何判断起点？如果num-1不在集合中，说明num是某个连续序列的起点。然后从这个起点开始，不断向后查找num+1, num+2...直到找不到为止。

这样每个序列只会被计算一次，时间复杂度O(n)。

**步骤：**
1. 将所有数字存入HashSet，方便O(1)查找
2. 遍历集合中的每个数字num：
   - 如果num-1不在集合中，说明num是某个序列的起点
   - 从num开始，不断查找num+1, num+2...是否在集合中
   - 统计连续序列的长度
   - 更新最大长度
3. 返回最大长度

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        // 用集合存储所有数字，方便快速查找
        Set<Integer> set = new HashSet<>();
        int maxLen = 0;
        for (int num : nums) {
            set.add(num);
        }
        for (int num : set) {
            // 只从序列的起点开始计算（前一个数字不在集合中）
            if(!set.contains(num-1)){
                int currLen = 1;
                int currNUm = num;
                // 向后查找连续的数字
                while(set.contains(currNUm+1)){
                    currNUm++;
                    currLen++;
                }
                maxLen = Math.max(maxLen, currLen);
            }
        }
        return maxLen;
    }
}
```

## 移动零
**思路：**
这道题要求将所有0移动到数组末尾，同时保持非零元素的相对顺序。不能使用额外空间，需要在原数组上操作。

使用双指针技巧：一个指针记录下一个非零元素应该放置的位置，另一个指针遍历整个数组。当遇到非零元素时，将其放到前面，然后两个指针都向前移动。

**步骤：**
1. 初始化一个指针zeroIndex，表示下一个非零元素应该放置的位置（初始为0）
2. 遍历数组，对于每个元素：
   - 如果当前元素不为0，将其放到zeroIndex位置，然后zeroIndex++
   - 如果当前元素为0，继续遍历
3. 遍历完成后，从zeroIndex开始到数组末尾，将所有位置填充为0

```java
class Solution {
    public void moveZeroes(int[] nums) {
        // zeroIndex: 下一个非零元素应该放的位置
        int zeroIndex = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                // 将非零元素移到前面
                nums[zeroIndex++] = nums[i];
            }
        }
        // 剩余位置全部填0
        while(zeroIndex < nums.length) {
            nums[zeroIndex++] = 0;
        }
    }
}
```

## 最多水的容器
**思路：**
这道题要求找到两条线，使得它们与x轴围成的容器能容纳最多的水。面积 = 宽度 × 较小的高度。

暴力解法需要O(n²)时间。我们可以用双指针优化到O(n)。

关键观察：如果移动高度较大的指针，面积一定不会增大（因为宽度减小，高度不变或变小）。所以我们应该移动高度较小的指针，这样才有可能找到更大的面积。

**步骤：**
1. 初始化两个指针left和right，分别指向数组两端
2. 计算当前面积：min(height[left], height[right]) × (right - left)
3. 更新最大面积
4. 比较两个指针指向的高度：
   - 如果left高度小，left++
   - 如果right高度小，right--
   - 如果相等，可以移动任意一个
5. 重复步骤2-4，直到left >= right
6. 返回最大面积

```java
class Solution {
    public int maxArea(int[] height) {
        int maxArea = 0;
        int left =  0;
        int right = height.length -1;
        while (left < right) {
            // 计算当前面积：宽度 * 较小的高度
            int currArea = (right - left) * Math.min(height[left], height[right]);
            maxArea = Math.max(currArea, maxArea);
            // 移动高度较小的指针
            if (height[left] < height[right]) left++;
            else right--;
        }
        return maxArea;
    }
}
```

## 三数之和
**思路：**
这道题要求找出所有不重复的三元组，使得它们的和为0。关键是要避免重复结果。

先对数组排序，然后固定第一个数，用双指针在剩余部分找另外两个数。排序后可以方便地使用双指针，也方便去重。

**步骤：**
1. 对数组进行排序
2. 遍历数组，固定第一个数nums[i]：
   - 如果nums[i] > 0，直接break（因为数组已排序，后面都是正数，不可能和为0）
   - 如果i > 0且nums[i] == nums[i-1]，跳过（去重）
   - 设置left = i+1, right = n-1，target = -nums[i]
   - 当left < right时：
     - 计算sum = nums[left] + nums[right]
     - 如果sum == target，找到一组解，加入结果
       - 然后跳过所有相同的left和right值（去重）
       - left++, right--
     - 如果sum < target，left++（需要更大的数）
     - 如果sum > target，right--（需要更小的数）
3. 返回所有结果

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        Arrays.sort(nums);
        List<List<Integer>> result = new ArrayList<>();
        for (int i = 0; i < nums.length - 2; i++) {
            // 如果最小的数都大于0，不可能有三数之和为0
            if (nums[i] > 0) break;
            // 去重：跳过相同的固定数
            if (i > 0 && nums[i] == nums[i - 1]) continue;
            int left = i + 1; 
            int right = nums.length - 1;
            int target = -nums[i];
            while (left < right) {
                int sum = nums[left] + nums[right];
                if (sum == target) {
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    // 去重：跳过相同的左指针值
                    while(left< right && nums[left] == nums[left+1]) left++;
                    // 去重：跳过相同的右指针值
                    while(left< right && nums[right] == nums[right-1]) right--;
                    left++; 
                    right--;
                }else if (sum < target) {
                    left++; // 太小，左指针右移
                } else {
                    right--; // 太大，右指针左移
                }
            }
        }
        return result;
    }
}
```

## 无重复字符的最长子串
**思路：**
这道题要求找出不包含重复字符的最长子串的长度。使用滑动窗口（双指针）技巧。

维护一个窗口[left, right]，用HashSet记录窗口内的字符。右指针不断扩展，如果遇到重复字符，左指针收缩直到窗口内没有重复字符。在扩展过程中记录窗口的最大长度。

**步骤：**
1. 初始化left = 0，max = 0，创建一个HashSet记录窗口内的字符
2. 遍历字符串，right从0到n-1：
   - 如果当前字符s[right]在集合中（重复）：
     - 不断移除s[left]并left++，直到s[right]不在集合中
   - 将s[right]加入集合
   - 更新最大长度：max = max(max, right - left + 1)
3. 返回max

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int left = 0;
        int max = 0;
        // 用集合记录窗口内的字符
        Set<Character> set = new HashSet<>();
        for (int right = 0; right < s.length(); right++) {
            // 如果遇到重复字符，收缩左边界
            while (set.contains(s.charAt(right))) {
                set.remove(s.charAt(left));
                left++;
            }
            // 将当前字符加入窗口
            set.add(s.charAt(right));
            // 更新最大长度
            max = Math.max(max, right - left + 1);
        }
        return max;
    }
}
```

## 找到字符串中所有字母异位词
**思路：**
这道题要求找出字符串s中所有p的异位词的起始索引。异位词是指字母相同但排列不同的字符串。

使用滑动窗口+字符计数。因为窗口大小固定（等于p的长度），我们可以用固定大小的窗口在s上滑动，每次比较窗口内的字符频次是否和p相同。

**步骤：**
1. 如果s长度小于p长度，返回空列表
2. 创建两个数组parray和sarray，分别记录p和窗口的字符频次（26个小写字母）
3. 初始化第一个窗口：统计p的字符频次，同时统计s前plen个字符的频次
4. 检查第一个窗口是否匹配，如果匹配，加入索引0
5. 滑动窗口：从索引plen开始遍历s：
   - 将新字符s[i]加入窗口（sarray[s[i]-'a']++）
   - 将旧字符s[i-plen]移出窗口（sarray[s[i-plen]-'a']--）
   - 比较当前窗口的字符频次是否和p相同
   - 如果相同，加入起始索引i-plen+1
6. 返回所有匹配的起始索引

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        List<Integer> res = new ArrayList();
        int slen = s.length();
        int plen = p.length();
        if (slen < plen) return res;
        // 用数组记录字符频次（26个小写字母）
        int[] parray = new int[26];
        int[] sarray = new int[26];
        // 统计p的字符频次，同时初始化第一个窗口
        for (int i = 0; i < plen; i++) {
            parray[p.charAt(i) - 'a']++;  
            sarray[s.charAt(i) - 'a']++;  
        }
        // 检查第一个窗口是否匹配
        if (Arrays.equals(sarray, parray)) {
            res.add(0);
        }
        // 滑动窗口
        for (int i = plen; i< slen;i ++){
            // 加入新字符
            sarray[s.charAt(i) - 'a']++;
            // 删除旧字符
            sarray[s.charAt(i - plen) - 'a']--;
            // 检查当前窗口是否匹配
            if (Arrays.equals(sarray, parray)) {
                res.add(i - plen + 1);
            }
        }
        return res;
    }
}
```

## 和为key的子数组
**思路：**
这道题要求找出所有和为k的连续子数组的个数。

关键观察：如果前缀和sum[i] - sum[j] = k，那么子数组[j+1, i]的和就是k。所以我们可以用哈希表记录每个前缀和出现的次数，当遍历到位置i时，查找是否存在前缀和等于currentSum - k。

**步骤：**
1. 创建哈希表prefixSumCount，key是前缀和，value是出现次数
2. 初始化：prefixSumCount.put(0, 1)，表示前缀和为0（空子数组）出现1次
3. 初始化currentSum = 0, count = 0
4. 遍历数组，对于每个数字num：
   - 更新当前前缀和：currentSum += num
   - 计算需要的前缀和：need = currentSum - k
   - 如果need在哈希表中，count += prefixSumCount.get(need)
   - 将当前前缀和加入哈希表（或更新出现次数）
5. 返回count

```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        // key: 前缀和, value: 出现次数
        Map<Integer, Integer> prefixSumCount = new HashMap<>();
        prefixSumCount.put(0, 1); // 前缀和为 0 的情况（空子数组）
        int currentSum = 0;
        int count = 0;
        for (int num : nums) {
            currentSum += num;
            // 需要的前缀和 = 当前前缀和 - k
            int need = currentSum - k;
            // 如果之前有前缀和等于 need，说明可以构成和为 k 的子数组
            if (prefixSumCount.containsKey(need)) {
                count += prefixSumCount.get(need);
            }
            // 更新当前前缀和的出现次数
            prefixSumCount.put(currentSum, prefixSumCount.getOrDefault(currentSum, 0) + 1);
        }
        return count;
    }
}
```

## 最大子数组和
**思路：**
这道题要求找出和最大的连续子数组。使用动态规划（Kadane算法）。

核心思想：对于每个位置i，以i结尾的最大子数组和，要么是继续之前的子数组（currentSum + nums[i]），要么是重新开始（nums[i]）。我们取两者中的较大值。

**步骤：**
1. 初始化currentSum = nums[0]，表示以当前位置结尾的最大子数组和
2. 初始化maxSum = nums[0]，表示全局最大子数组和
3. 从索引1开始遍历数组：
   - 更新currentSum = max(nums[i], currentSum + nums[i])
     - 如果currentSum + nums[i] > nums[i]，继续之前的子数组
     - 否则，重新开始
   - 更新maxSum = max(maxSum, currentSum)
4. 返回maxSum

```java
class Solution {
    public int maxSubArray(int[] nums) {
        // currentSum: 以当前位置结尾的最大子数组和
        int currentSum = nums[0];
        int maxSum = nums[0];
        for (int i = 1; i < nums.length; i++) {
            // 要么继续之前的子数组，要么重新开始
            currentSum = Math.max(nums[i],nums[i] + currentSum);
            maxSum = Math.max(currentSum, maxSum);
        }
        return maxSum;
    }
}
```

## 合并区间
**思路：**
这道题要求合并所有重叠的区间。关键是要先排序，然后判断相邻区间是否重叠。

如果两个区间重叠，那么前一个区间的右端点一定大于等于后一个区间的左端点。合并时，取两个区间右端点的最大值作为合并后的右端点。

**步骤：**
1. 如果区间数组为空，返回空数组
2. 按区间的左端点排序
3. 创建结果列表merge
4. 遍历排序后的区间：
   - 如果merge为空，或者当前区间的左端点 > 已合并区间的最右端点：
     - 说明不重叠，直接添加当前区间
   - 否则：
     - 说明重叠，更新已合并区间的最右端点 = max(当前右端点, 已合并区间的右端点)
5. 将结果列表转为数组返回

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        if(intervals.length == 0){
            return new int[0][2];
        }
        // 按左端点排序
        Arrays.sort(intervals,new Comparator<int[]>(){
            public int compare(int[] inter1, int[] inter2){
                return inter1[0] - inter2[0];
            }
        });
        List<int[]> merge = new ArrayList<int[]>();
        for(int[] inter : intervals){
            int L = inter[0],R = inter[1];
            // 如果当前区间左端点在已合并区间最右端点之后，不重叠
            if(merge.size()==0 || L>merge.get(merge.size()-1)[1]){
                merge.add(new int[]{L,R});
            }else{
                // 合并：更新最右端点
                merge.get(merge.size()-1)[1] = Math.max(R,merge.get(merge.size()-1)[1]);
            }
        }
        return merge.toArray(new int[merge.size()][]);
    }
}
```

## 轮转数组
**思路：**
这道题要求将数组向右轮转k个位置。要求使用O(1)额外空间。

巧妙的方法：先整体反转数组，然后分别反转前k个元素和后n-k个元素。这样就能实现向右轮转k步的效果。

例如：[1,2,3,4,5] 向右轮转2步
- 整体反转：[5,4,3,2,1]
- 反转前2个：[4,5,3,2,1]
- 反转后3个：[4,5,1,2,3]

**步骤：**
1. 处理k：k = k % n（因为轮转n次等于不转）
2. 反转整个数组：reverse(nums, 0, n-1)
3. 反转前k个元素：reverse(nums, 0, k-1)
4. 反转后n-k个元素：reverse(nums, k, n-1)

```java
class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        k = k % n; // 处理k大于数组长度的情况
        // 先整体反转
        reverse(nums, 0, n - 1);
        // 反转前k个元素
        reverse(nums, 0, k - 1);
        // 反转后n-k个元素
        reverse(nums, k, n - 1);
    }
    
    private void reverse(int[] nums, int start, int end) {
        while (start < end) {
            int temp = nums[start];
            nums[start] = nums[end];
            nums[end] = temp;
            start++;
            end--;
        }
    }
}
```

## 除自身以外数组的乘积
**思路：**
这道题要求返回一个数组，其中每个元素是除自身外所有元素的乘积。要求O(n)时间，且不能使用除法。

核心思想：对于位置i，结果 = 左边所有元素的乘积 × 右边所有元素的乘积。

我们可以先从左到右计算每个位置左边所有元素的乘积，然后从右到左计算右边所有元素的乘积，并直接乘到结果中。

**步骤：**
1. 创建结果数组result
2. 第一次遍历（从左到右）：
   - result[0] = 1
   - result[i] = result[i-1] * nums[i-1]（左边所有元素的乘积）
3. 第二次遍历（从右到左）：
   - 用变量right记录右边所有元素的乘积（初始为1）
   - result[i] *= right
   - right *= nums[i]
4. 返回result

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n = nums.length;
        int[] result = new int[n];
        // 先计算每个位置左边所有元素的乘积
        result[0] = 1;
        for (int i = 1; i < n; i++) {
            result[i] = result[i - 1] * nums[i - 1];
        }
        // 再计算右边所有元素的乘积，并直接乘到结果中
        int right = 1;
        for (int i = n - 1; i >= 0; i--) {
            result[i] *= right;
            right *= nums[i];
        }
        return result;
    }
}
```

## 矩阵置0
**思路：**
这道题要求如果矩阵中某个元素为0，则将其所在的行和列都置为0。要求O(1)额外空间。

关键思想：用第一行和第一列来标记哪些行和列需要置0。但这样会覆盖第一行和第一列本身的信息，所以需要先记录第一行和第一列是否有0。

**步骤：**
1. 检查第一行是否有0，记录到firstRowZero
2. 检查第一列是否有0，记录到firstColZero
3. 遍历矩阵（从第二行第二列开始）：
   - 如果matrix[i][j] == 0：
     - 将matrix[i][0] = 0（标记第i行需要置0）
     - 将matrix[0][j] = 0（标记第j列需要置0）
4. 根据标记置0（从第二行第二列开始）：
   - 如果matrix[i][0] == 0或matrix[0][j] == 0，将matrix[i][j] = 0
5. 如果firstRowZero为true，将第一行全部置0
6. 如果firstColZero为true，将第一列全部置0

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        boolean firstRowZero = false;
        boolean firstColZero = false;
        // 检查第一行是否有0
        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                firstRowZero = true;
                break;
            }
        }
        // 检查第一列是否有0
        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                firstColZero = true;
                break;
            }
        }
        // 用第一行和第一列标记
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }
        // 根据标记置0
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }
        // 处理第一行
        if (firstRowZero) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }
        // 处理第一列
        if (firstColZero) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }
}
```

## 螺旋矩阵
**思路：**
这道题要求按顺时针螺旋顺序返回矩阵中的所有元素。

使用四个边界变量（top, bottom, left, right）控制遍历范围。按顺时针方向：从左到右、从上到下、从右到左、从下到上。每完成一个方向，就缩小对应的边界。

**步骤：**
1. 初始化四个边界：top=0, bottom=m-1, left=0, right=n-1
2. 当top <= bottom && left <= right时循环：
   - 从左到右：遍历top行，从left到right，加入结果
   - top++（缩小上边界）
   - 从上到下：遍历right列，从top到bottom，加入结果
   - right--（缩小右边界）
   - 如果top <= bottom（还有行）：
     - 从右到左：遍历bottom行，从right到left，加入结果
     - bottom--（缩小下边界）
   - 如果left <= right（还有列）：
     - 从下到上：遍历left列，从bottom到top，加入结果
     - left++（缩小左边界）
3. 返回结果

```java
class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        List<Integer> result = new ArrayList<>();
        if (matrix == null || matrix.length == 0) return result;
        int top = 0, bottom = matrix.length - 1;
        int left = 0, right = matrix[0].length - 1;
        while (top <= bottom && left <= right) {
            // 从左到右
            for (int j = left; j <= right; j++) {
                result.add(matrix[top][j]);
            }
            top++;
            // 从上到下
            for (int i = top; i <= bottom; i++) {
                result.add(matrix[i][right]);
            }
            right--;
            // 从右到左（需要检查是否还有行）
            if (top <= bottom) {
                for (int j = right; j >= left; j--) {
                    result.add(matrix[bottom][j]);
                }
                bottom--;
            }
            // 从下到上（需要检查是否还有列）
            if (left <= right) {
                for (int i = bottom; i >= top; i--) {
                    result.add(matrix[i][left]);
                }
                left++;
            }
        }
        return result;
    }
}
```

## 旋转图像
**思路：**
这道题要求将n×n矩阵顺时针旋转90度，要求原地修改。

巧妙的方法：先沿主对角线转置矩阵，然后每行反转。这样就能实现顺时针旋转90度。

例如：[[1,2,3],[4,5,6],[7,8,9]]
- 转置后：[[1,4,7],[2,5,8],[3,6,9]]
- 每行反转：[[7,4,1],[8,5,2],[9,6,3]]

**步骤：**
1. 沿主对角线转置：
   - 遍历i从0到n-1，j从i到n-1
   - 交换matrix[i][j]和matrix[j][i]
2. 每行反转：
   - 遍历每一行，使用双指针反转该行

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        // 先沿主对角线转置
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        // 然后每行反转
        for (int i = 0; i < n; i++) {
            int left = 0, right = n - 1;
            while (left < right) {
                int temp = matrix[i][left];
                matrix[i][left] = matrix[i][right];
                matrix[i][right] = temp;
                left++;
                right--;
            }
        }
    }
}
```

## 搜索二维矩阵
**思路：**
这道题要求在一个每行从左到右递增、每列从上到下递增的矩阵中查找目标值。

可以将二维矩阵看作一维有序数组，使用二分查找。关键是坐标转换：一维索引mid对应的二维坐标是(row, col)，其中row = mid / n, col = mid % n。

**步骤：**
1. 初始化left = 0, right = m*n - 1
2. 当left <= right时：
   - 计算mid = left + (right - left) / 2
   - 将一维索引转换为二维坐标：
     - row = mid / n
     - col = mid % n
   - 如果matrix[row][col] == target，返回true
   - 如果matrix[row][col] < target，left = mid + 1
   - 否则，right = mid - 1
3. 返回false

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length;
        int n = matrix[0].length;
        int left = 0, right = m * n - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            // 将一维索引转换为二维坐标
            int row = mid / n;
            int col = mid % n;
            if (matrix[row][col] == target) {
                return true;
            } else if (matrix[row][col] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return false;
    }
}
```

## 相交链表
**思路：**
这道题要求找出两个链表的相交节点。如果两个链表不相交，返回null。

巧妙的方法：用两个指针分别从两个链表头开始遍历，当一个指针到达末尾时，让它从另一个链表头开始。这样两个指针会在相交点相遇（如果相交的话），或者同时到达null（不相交）。

原理：假设链表A长度为a，链表B长度为b，相交部分长度为c。指针A走完A后走B，指针B走完B后走A，它们都会走a+b-c步，在相交点相遇。

**步骤：**
1. 初始化两个指针pA和pB，分别指向headA和headB
2. 当pA != pB时循环：
   - 如果pA为null，pA = headB；否则pA = pA.next
   - 如果pB为null，pB = headA；否则pB = pB.next
3. 返回pA（此时pA == pB，要么是相交点，要么是null）

```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode pA = headA;
        ListNode pB = headB;
        // 两个指针分别遍历，如果到达末尾就切换到另一个链表
        // 这样会在相交点相遇，或者同时为null（不相交）
        while (pA != pB) {
            pA = pA == null ? headB : pA.next;
            pB = pB == null ? headA : pB.next;
        }
        return pA;
    }
}
```

## 反转链表
**思路：**
这道题要求反转一个单链表。可以使用迭代或递归方法。

迭代方法：使用三个指针prev、curr、next。每次将curr的next指向prev，然后三个指针都向前移动，直到curr为null。

**步骤：**
1. 初始化prev = null, curr = head
2. 当curr != null时循环：
   - 保存下一个节点：next = curr.next
   - 反转当前节点的指针：curr.next = prev
   - 移动指针：prev = curr, curr = next
3. 返回prev（新的头节点）

```java
class Solution {

        ListNode curr = head;

            // 保存下一个节点
            ListNode next = curr.next;
            // 反转当前节点的指针
            curr.next = prev;
            // 移动指针
            prev = curr;
            curr = next;
        }
        return prev;
    }
}
```

## 回文链表
**思路：**
这道题要求判断一个链表是否为回文链表。要求O(n)时间和O(1)空间。

关键思想：找到链表中点，反转后半部分，然后比较前半部分和反转后的后半部分是否相同。注意处理奇数长度和偶数长度的情况。

**步骤：**
1. 如果链表为空或只有一个节点，返回true
2. 使用快慢指针找到中点：
   - slow和fast都从head开始
   - slow每次走1步，fast每次走2步
   - 当fast到达末尾时，slow指向中点（或中点前一个节点）
3. 反转后半部分：从slow.next开始反转
4. 比较前半部分和反转后的后半部分：
   - firstHalf从head开始，secondHalf从反转后的头开始
   - 逐个比较节点值
   - 如果都相同，返回true；否则返回false

```java
class Solution {
    public boolean isPalindrome(ListNode head) {
        if (head == null || head.next == null) return true;
        // 找到中点
        ListNode slow = head, fast = head;
        while (fast.next != null && fast.next.next != null) {
            slow = slow.next;
            fast = fast.next.next;
        }
        // 反转后半部分
        ListNode secondHalf = reverse(slow.next);
        // 比较前半部分和反转后的后半部分
        ListNode firstHalf = head;
        while (secondHalf != null) {
            if (firstHalf.val != secondHalf.val) {
                return false;
            }
            firstHalf = firstHalf.next;
            secondHalf = secondHalf.next;
        }
        return true;
    }
    
    private ListNode reverse(ListNode head) {
        ListNode prev = null;
        ListNode curr = head;
        while (curr != null) {
            ListNode next = curr.next;
            curr.next = prev;
            prev = curr;
            curr = next;
        }
        return prev;
    }
}
```

## 环形链表
**思路：**
这道题要求判断链表中是否有环。要求O(1)空间。

使用快慢指针（Floyd判圈算法）：快指针每次走两步，慢指针每次走一步。如果有环，快指针会追上慢指针；如果没有环，快指针会先到达null。

**步骤：**
1. 如果链表为空或只有一个节点，返回false
2. 初始化slow = head, fast = head.next
3. 当slow != fast时循环：
   - 如果fast == null或fast.next == null，说明没有环，返回false
   - slow = slow.next
   - fast = fast.next.next
4. 如果退出循环，说明有环，返回true

```java
public class Solution {
    public boolean hasCycle(ListNode head) {
        if (head == null || head.next == null) return false;
        ListNode slow = head;
        ListNode fast = head.next;
        while (slow != fast) {
            // 快指针到达末尾，说明没有环
            if (fast == null || fast.next == null) {
                return false;
            }
            slow = slow.next;
            fast = fast.next.next;
        }
        return true;
    }
}
```

## 环形链表起始点
**思路：**
这道题要求找出环形链表中环的入口节点。如果没有环，返回null。

使用快慢指针：先用快慢指针找到相遇点，然后一个指针从头开始，一个从相遇点开始，同时移动，相遇点就是环的入口。

原理：假设从head到入口距离为a，从入口到相遇点距离为b，环的长度为c。快指针走了a+b+c，慢指针走了a+b。因为快指针速度是慢指针的2倍，所以2(a+b) = a+b+c，即a = c-b。所以从head和相遇点同时出发，会在入口相遇。

**步骤：**
1. 使用快慢指针找到相遇点：
   - slow和fast都从head开始
   - slow每次走1步，fast每次走2步
   - 当slow == fast时，找到相遇点
2. 如果fast == null或fast.next == null，说明没有环，返回null
3. 找环的入口：
   - ptr从head开始，slow从相遇点开始
   - 同时移动，每次各走1步
   - 当ptr == slow时，就是环的入口
4. 返回ptr

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode slow = head;
        ListNode fast = head;
        // 找到相遇点
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
            if (slow == fast) {
                // 找到相遇点，开始找入口
                ListNode ptr = head;
                while (ptr != slow) {
                    ptr = ptr.next;
                    slow = slow.next;
                }
                return ptr;
            }
        }
        return null;
    }
}
```

## 合并两个有序链表

```java
class Solution {

                curr.next = list2;
```

5. 删除节点：slow.next = slow.next.next
6. 返回dummy.next

```java
class Solution {

        for (int i = 0; i <= n; i++) {

            slow = slow.next;
```

## 随机链表复制
**思路：**
这道题要求深拷贝一个包含随机指针的链表。难点是random指针可能指向任意节点。

巧妙的方法：分三步完成。第一步：在每个原节点后面插入一个复制节点；第二步：设置random指针（复制节点的random = 原节点的random.next）；第三步：拆分两个链表。

2. 第一步：在每个节点后面插入复制节点：

     - curr.next = copy

3. 第二步：设置random指针：
   - 遍历原链表（每次跳两步），对每个节点curr：
     - 如果curr.random != null，curr.next.random = curr.random.next
     - curr = curr.next.next
4. 第三步：拆分两个链表：
   - 保存新链表的头：newHead = head.next
   - 遍历原链表，恢复原链表并构建新链表
5. 返回newHead

```java
class Solution {
    public Node copyRandomList(Node head) {
        if (head == null) return null;
        // 第一步：在每个节点后面插入复制节点
        Node curr = head;
        while (curr != null) {
            Node copy = new Node(curr.val);
            copy.next = curr.next;
            curr.next = copy;
            curr = copy.next;
        }
        // 第二步：设置random指针
        curr = head;
        while (curr != null) {
            if (curr.random != null) {
                curr.next.random = curr.random.next;
            }
            curr = curr.next.next;
        }
        // 第三步：拆分两个链表
        curr = head;
        Node newHead = head.next;
        while (curr != null) {
            Node copy = curr.next;
            curr.next = copy.next;
            if (copy.next != null) {
                copy.next = copy.next.next;
            }
            curr = curr.next;
        }
        return newHead;
    }
}
```

## 排序链表
**思路：**
这道题要求对链表进行排序，要求O(n log n)时间复杂度和O(1)空间复杂度。

使用归并排序：先找到链表中点，然后递归排序左右两部分，最后合并两个有序链表。这是链表排序的标准方法。

**步骤：**
1. 如果head == null或head.next == null，返回head
2. 找到中点：
   - 使用快慢指针，slow和fast都从head开始
   - slow每次走1步，fast每次走2步
   - 当fast到达末尾时，slow指向中点前一个节点
3. 分割链表：
   - mid = slow.next
   - slow.next = null（断开连接）
4. 递归排序：
   - left = sortList(head)
   - right = sortList(mid)
5. 合并两个有序链表，返回合并后的头节点

```java
class Solution {
    public ListNode sortList(ListNode head) {
        if (head == null || head.next == null) {
            return head;
        }
        // 找到中点
        ListNode slow = head, fast = head.next;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
        }
        // 分割链表
        ListNode mid = slow.next;
        slow.next = null;
        // 递归排序
        ListNode left = sortList(head);
        ListNode right = sortList(mid);
        // 合并
        return merge(left, right);
    }
    
    private ListNode merge(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0);
        ListNode curr = dummy;
        while (l1 != null && l2 != null) {
            if (l1.val <= l2.val) {
                curr.next = l1;
                l1 = l1.next;
            } else {
                curr.next = l2;
                l2 = l2.next;
            }
            curr = curr.next;
        }
        curr.next = l1 != null ? l1 : l2;
        return dummy.next;
    }
}
```

## LRU缓存
**思路：**
这道题要求实现LRU（Least Recently Used）缓存。需要支持get和put操作，时间复杂度都是O(1)。

使用哈希表+双向链表：哈希表实现O(1)查找，双向链表实现O(1)的插入和删除。最近使用的节点放在头部，最久未使用的在尾部。当容量满时，删除尾部节点。

**步骤：**
1. 定义双向链表节点Node，包含key、value、prev、next
2. 初始化：
   - 创建虚拟头尾节点head和tail
   - head.next = tail, tail.prev = head
   - 创建哈希表map存储key到Node的映射
3. get操作：
   - 从map中获取节点
   - 如果不存在，返回-1
   - 如果存在，将节点移到头部，返回value
4. put操作：
   - 如果key已存在，更新value并移到头部
   - 如果key不存在：
     - 如果容量已满，删除尾部节点
     - 创建新节点，加入头部和map
5. 辅助方法：
   - addToHead：将节点加入头部
   - removeNode：删除节点
   - moveToHead：将节点移到头部（先删除再加入头部）

```java
class LRUCache {
    class Node {
        int key, value;
        Node prev, next;
        Node(int key, int value) {
            this.key = key;
            this.value = value;
        }
    }
    
    private int capacity;
    private Map<Integer, Node> map;
    private Node head, tail;
    
    public LRUCache(int capacity) {
        this.capacity = capacity;
        this.map = new HashMap<>();
        // 创建虚拟头尾节点
        head = new Node(0, 0);
        tail = new Node(0, 0);
        head.next = tail;
        tail.prev = head;
    }
    
    public int get(int key) {
        Node node = map.get(key);
        if (node == null) return -1;
        // 移动到头部
        moveToHead(node);
        return node.value;
    }
    
    public void put(int key, int value) {
        Node node = map.get(key);
        if (node != null) {
            node.value = value;
            moveToHead(node);
        } else {
            node = new Node(key, value);
            if (map.size() >= capacity) {
                // 删除尾部节点
                Node last = tail.prev;
                removeNode(last);
                map.remove(last.key);
            }
            addToHead(node);
            map.put(key, node);
        }
    }
    
    private void addToHead(Node node) {
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }
    
    private void removeNode(Node node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }
    
    private void moveToHead(Node node) {
        removeNode(node);
        addToHead(node);
    }
}
```

## 二叉树中序遍历

可以使用递归或迭代方法。递归方法更直观：先递归遍历左子树，然后访问根节点，最后递归遍历右子树。

**步骤：**

2. 递归遍历左子树
3. 访问根节点（将值加入结果）

```java
class Solution {
```

## 二叉树最大深度
**思路：**
这道题要求计算二叉树的最大深度。最大深度是指从根节点到最远叶子节点的最长路径上的节点数。

使用递归：树的最大深度 = 1 + max(左子树深度, 右子树深度)。递归的终止条件是节点为null，此时深度为0。

**步骤：**
1. 如果根节点为null，返回0
2. 递归计算左子树的最大深度
3. 递归计算右子树的最大深度
4. 返回1 + max(左子树深度, 右子树深度)

```java
class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) {
            return 0;
        }
        // 左子树深度和右子树深度的最大值 + 1
        return 1 + Math.max(maxDepth(root.left), maxDepth(root.right));
    }
}
```

## 对称二叉树
**思路：**
这道题要求判断一棵二叉树是否是对称的（镜像对称）。

使用递归：判断左子树和右子树是否镜像对称。两个树镜像对称的条件是：它们的根节点值相等，且左树的左子树和右树的右子树镜像对称，左树的右子树和右树的左子树镜像对称。

**步骤：**
1. 如果根节点为null，返回true
2. 调用辅助函数isMirror(root.left, root.right)
3. isMirror函数：
   - 如果两个节点都为null，返回true
   - 如果只有一个为null，返回false
   - 如果值不相等，返回false
   - 递归判断：left.left和right.right是否对称，left.right和right.left是否对称

```java
class Solution {
    public boolean isSymmetric(TreeNode root) {
        if (root == null) return true;
        return isMirror(root.left, root.right);
    }
    
    private boolean isMirror(TreeNode left, TreeNode right) {
        if (left == null && right == null) return true;
        if (left == null || right == null) return false;
        // 值相等且左的左和右的右对称，左的右和右的左对称
        return left.val == right.val 
            && isMirror(left.left, right.right) 
            && isMirror(left.right, right.left);
    }
}
```

## 二叉树的直径
**思路：**
这道题要求计算二叉树的直径。直径是指任意两个节点之间最长路径上的节点数。

关键观察：对于每个节点，经过它的最长路径 = 左子树深度 + 右子树深度。我们可以在计算深度的过程中，同时更新最大直径。

**步骤：**
1. 定义一个全局变量maxDiameter记录最大直径
2. 调用maxDepth函数计算深度，在计算过程中更新maxDiameter
3. maxDepth函数：
   - 如果节点为null，返回0
   - 递归计算左子树深度
   - 递归计算右子树深度
   - 更新maxDiameter = max(maxDiameter, leftDepth + rightDepth)
   - 返回1 + max(leftDepth, rightDepth)
4. 返回maxDiameter

```java
class Solution {
    int maxDiameter = 0;
    
    public int diameterOfBinaryTree(TreeNode root) {
        maxDepth(root);
        return maxDiameter;
    }
    
    private int maxDepth(TreeNode root) {
        if (root == null) return 0;
        int leftDepth = maxDepth(root.left);
        int rightDepth = maxDepth(root.right);
        // 更新最大直径
        maxDiameter = Math.max(maxDiameter, leftDepth + rightDepth);
        return 1 + Math.max(leftDepth, rightDepth);
    }
}
```

## 二叉树层序遍历
**思路：**
这道题要求对二叉树进行层序遍历（广度优先遍历），返回每一层的节点值。

使用队列：每次处理一层，将当前层的节点值加入结果，然后将下一层的节点加入队列。需要记录每层的节点数量，以便区分不同层。

**步骤：**
1. 如果根节点为null，返回空列表
2. 创建队列，将根节点入队
3. 当队列不为空时循环：
   - 记录当前层的节点数量size
   - 创建当前层的列表level
   - 循环size次：
     - 出队一个节点
     - 将节点值加入level
     - 如果节点有左子节点，入队
     - 如果节点有右子节点，入队
   - 将level加入结果
4. 返回结果

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            int size = queue.size();
            List<Integer> level = new ArrayList<>();
            // 处理当前层
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                level.add(node.val);
                // 加入下一层节点
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
            result.add(level);
        }
        return result;
    }
}
```

## 有序数组转为二叉搜索树
**思路：**
这道题要求将一个有序数组转换为高度平衡的二叉搜索树。

使用递归：每次选择中间元素作为根节点，左边构建左子树，右边构建右子树。这样可以保证树是平衡的，因为左右子树的节点数量最多相差1。

**步骤：**
1. 如果left > right，返回null
2. 计算中间位置：mid = left + (right - left) / 2
3. 创建根节点，值为nums[mid]
4. 递归构建左子树：root.left = buildBST(nums, left, mid - 1)
5. 递归构建右子树：root.right = buildBST(nums, mid + 1, right)
6. 返回根节点

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        return buildBST(nums, 0, nums.length - 1);
    }
    
    private TreeNode buildBST(int[] nums, int left, int right) {
        if (left > right) return null;
        // 选择中间元素作为根节点
        int mid = left + (right - left) / 2;
        TreeNode root = new TreeNode(nums[mid]);
        // 递归构建左右子树
        root.left = buildBST(nums, left, mid - 1);
        root.right = buildBST(nums, mid + 1, right);
        return root;
    }
}
```

## 验证二叉搜索树
**思路：**
这道题要求验证一棵二叉树是否是有效的二叉搜索树。

使用递归：每个节点都有一个取值范围[min, max]。对于根节点，范围是(-∞, +∞)。对于左子节点，范围是(min, root.val)。对于右子节点，范围是(root.val, max)。如果节点值不在范围内，就不是有效的BST。

**步骤：**
1. 调用辅助函数isValid(root, Long.MIN_VALUE, Long.MAX_VALUE)
2. isValid函数：
   - 如果节点为null，返回true
   - 如果节点值 <= min或 >= max，返回false
   - 递归检查左子树：isValid(root.left, min, root.val)
   - 递归检查右子树：isValid(root.right, root.val, max)
   - 返回两个递归结果的与

```java
class Solution {
    public boolean isValidBST(TreeNode root) {
        return isValid(root, Long.MIN_VALUE, Long.MAX_VALUE);
    }
    
    private boolean isValid(TreeNode root, long min, long max) {
        if (root == null) return true;
        // 当前节点值必须在范围内
        if (root.val <= min || root.val >= max) {
            return false;
        }
        // 左子树的值要小于根节点，右子树的值要大于根节点
        return isValid(root.left, min, root.val) 
            && isValid(root.right, root.val, max);
    }
}
```

4. 返回result

```java
class Solution {
```

## 二叉树的右视图
**思路：**
这道题要求返回二叉树的右视图，即从右侧看二叉树时能看到的节点值。

使用层序遍历：每层最右边的节点（即每层最后一个节点）就是右视图的节点。在层序遍历时，记录每层的最后一个节点即可。

**步骤：**
1. 如果根节点为null，返回空列表
2. 创建队列，将根节点入队
3. 当队列不为空时循环：
   - 记录当前层的节点数量size
   - 循环size次：
     - 出队一个节点
     - 如果i == size - 1（最后一个节点），将节点值加入结果
     - 如果节点有左子节点，入队
     - 如果节点有右子节点，入队
4. 返回结果

```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        if (root == null) return result;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            int size = queue.size();
            // 每层最后一个节点就是右视图的节点
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                if (i == size - 1) {
                    result.add(node.val);
                }
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
        }
        return result;
    }
```

```java
class Solution {
```

## 从前序中序遍历序列构造二叉树
**思路：**
这道题要求根据前序遍历和中序遍历序列构造二叉树。

使用递归：前序遍历的第一个元素是根节点，在中序遍历中找到根节点，左边是左子树，右边是右子树。用哈希表记录中序遍历中每个值的位置，可以快速查找。

**步骤：**
1. 用哈希表记录中序遍历中每个值的位置
2. 调用build函数构建树
3. build函数：
   - 如果preStart > preEnd，返回null
   - 创建根节点，值为preorder[preStart]
   - 在中序遍历中找到根节点位置rootIndex
   - 计算左子树大小leftSize = rootIndex - inStart
   - 递归构建左子树：build(preorder, preStart+1, preStart+leftSize, inorder, inStart, rootIndex-1)
   - 递归构建右子树：build(preorder, preStart+leftSize+1, preEnd, inorder, rootIndex+1, inEnd)
   - 返回根节点

```java
class Solution {
    Map<Integer, Integer> map = new HashMap<>();
    
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        // 用哈希表记录中序遍历中每个值的位置
        for (int i = 0; i < inorder.length; i++) {
            map.put(inorder[i], i);
        }
        return build(preorder, 0, preorder.length - 1, 
                    inorder, 0, inorder.length - 1);
    }
    
    private TreeNode build(int[] preorder, int preStart, int preEnd,
                          int[] inorder, int inStart, int inEnd) {
        if (preStart > preEnd) return null;
        // 前序遍历第一个元素是根节点
        TreeNode root = new TreeNode(preorder[preStart]);
        // 在中序遍历中找到根节点位置
        int rootIndex = map.get(root.val);
        int leftSize = rootIndex - inStart;
        // 递归构建左右子树
        root.left = build(preorder, preStart + 1, preStart + leftSize,
                         inorder, inStart, rootIndex - 1);
        root.right = build(preorder, preStart + leftSize + 1, preEnd,
                          inorder, rootIndex + 1, inEnd);
        return root;
    }
}
```

## 路径总和三
**思路：**
这道题要求找出路径和等于目标值的路径数量。路径不需要从根节点开始，也不需要在叶子节点结束。

使用双重递归：外层递归遍历每个节点，内层递归计算以该节点为起点的路径和。对于每个节点，都要检查以它为起点的所有路径。

**步骤：**
1. 如果根节点为null，返回0
2. 从当前节点开始查找路径：findPath(root, targetSum)
3. 递归处理左右子树：pathSum(root.left, targetSum) + pathSum(root.right, targetSum)
4. findPath函数：
   - 如果节点为null，返回
   - 如果节点值等于targetSum，count++
   - 递归查找左子树：findPath(root.left, targetSum - root.val)
   - 递归查找右子树：findPath(root.right, targetSum - root.val)
5. 返回count

```java
class Solution {
    int count = 0;
    
    public int pathSum(TreeNode root, int targetSum) {
        if (root == null) return 0;
        // 从当前节点开始查找
        findPath(root, targetSum);
        // 递归处理左右子树
        pathSum(root.left, targetSum);
        pathSum(root.right, targetSum);
        return count;
    }
    
    private void findPath(TreeNode root, int targetSum) {
        if (root == null) return;
        if (root.val == targetSum) {
            count++;
        }
        // 继续向下查找
        findPath(root.left, targetSum - root.val);
        findPath(root.right, targetSum - root.val);
    }
}
```

## 二叉树的最近公共祖先
**思路：**
这道题要求找出二叉树中两个节点的最近公共祖先。

使用递归：如果当前节点是p或q，返回当前节点。递归查找左右子树，如果左右子树都找到了（都不为null），说明当前节点是公共祖先。如果只有一边找到，返回找到的那边。

**步骤：**
1. 如果root == null或root == p或root == q，返回root
2. 递归查找左子树：left = lowestCommonAncestor(root.left, p, q)
3. 递归查找右子树：right = lowestCommonAncestor(root.right, p, q)
4. 如果left != null && right != null，说明当前节点是公共祖先，返回root
5. 否则返回left != null ? left : right

```java
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if (root == null || root == p || root == q) {
            return root;
        }
        TreeNode left = lowestCommonAncestor(root.left, p, q);
        TreeNode right = lowestCommonAncestor(root.right, p, q);
        // 如果左右都找到了，说明当前节点是公共祖先
        if (left != null && right != null) {
            return root;
        }
        // 否则返回找到的那个
        return left != null ? left : right;
    }
}
```

1. 初始化count = 0
2. 遍历网格的每个位置：

     - 进行DFS/BFS，将所有相连的'1'标记为'0'
     - count++
3. 返回count
4. DFS函数：
   - 如果位置越界或grid[i][j] == '0'，返回
   - 将grid[i][j]标记为'0'

```java
class Solution {
    public int numIslands(char[][] grid) {

        for (int i = 0; i < grid.length; i++) {
            for (int j = 0; j < grid[0].length; j++) {
```

## 腐烂的橘子
**思路：**
这道题要求计算所有橘子腐烂所需的最少分钟数。每分钟，腐烂的橘子会让相邻的新鲜橘子变腐烂。

使用BFS：先把所有腐烂的橘子加入队列，然后进行BFS，每次处理一层（同一分钟腐烂的），将相邻的新鲜橘子变腐烂并加入队列。记录时间。

**步骤：**
1. 遍历网格，找到所有腐烂的橘子（值为2）加入队列，统计新鲜橘子数量
2. 如果新鲜橘子数量为0，返回0
3. 初始化minutes = 0，定义四个方向
4. 当队列不为空时循环：
   - 记录当前层的数量size
   - 循环size次：
     - 出队一个腐烂橘子
     - 检查四个方向的相邻位置
     - 如果是新鲜橘子，将其变腐烂，新鲜数量--，加入队列
   - 如果队列不为空，minutes++
5. 如果新鲜橘子数量为0，返回minutes；否则返回-1

```java
class Solution {
    public int orangesRotting(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;
        Queue<int[]> queue = new LinkedList<>();
        int freshCount = 0;
        // 找到所有腐烂的橘子
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 2) {
                    queue.offer(new int[]{i, j});
                } else if (grid[i][j] == 1) {
                    freshCount++;
                }
            }
        }
        if (freshCount == 0) return 0;
        int minutes = 0;
        int[][] directions = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
        while (!queue.isEmpty()) {
            int size = queue.size();
            for (int i = 0; i < size; i++) {
                int[] curr = queue.poll();
                for (int[] dir : directions) {
                    int x = curr[0] + dir[0];
                    int y = curr[1] + dir[1];
                    if (x >= 0 && x < m && y >= 0 && y < n && grid[x][y] == 1) {
                        grid[x][y] = 2;
                        freshCount--;
                        queue.offer(new int[]{x, y});
                    }
                }
            }
            if (!queue.isEmpty()) minutes++;
        }
        return freshCount == 0 ? minutes : -1;
    }
}
```

## 课程表
**思路：**
这道题要求判断能否完成所有课程的学习。这是一个拓扑排序问题，判断有向图是否有环。

使用BFS进行拓扑排序：统计每个节点的入度，从入度为0的节点开始，每次移除一个节点，更新其他节点的入度。如果所有节点都能被移除，说明没有环，可以完成所有课程。

**步骤：**
1. 构建邻接表和入度数组：
   - 对于每个先修关系[ai, bi]，表示学习ai前需要先学习bi
   - graph[bi].add(ai)，inDegree[ai]++
2. 将所有入度为0的节点加入队列
3. 当队列不为空时循环：
   - 出队一个节点，count++
   - 遍历该节点的所有邻接节点：
     - 将邻接节点的入度减1
     - 如果入度变为0，加入队列
4. 如果count == numCourses，返回true；否则返回false

```java
class Solution {
    public boolean canFinish(int numCourses, int[][] prerequisites) {
        // 构建邻接表和入度数组
        List<List<Integer>> graph = new ArrayList<>();
        int[] inDegree = new int[numCourses];
        for (int i = 0; i < numCourses; i++) {
            graph.add(new ArrayList<>());
        }
        for (int[] pre : prerequisites) {
            graph.get(pre[1]).add(pre[0]);
            inDegree[pre[0]]++;
        }
        // BFS：从入度为0的节点开始
        Queue<Integer> queue = new LinkedList<>();
        for (int i = 0; i < numCourses; i++) {
            if (inDegree[i] == 0) {
                queue.offer(i);
            }
        }
        int count = 0;
        while (!queue.isEmpty()) {
            int course = queue.poll();
            count++;
            for (int next : graph.get(course)) {
                inDegree[next]--;
                if (inDegree[next] == 0) {
                    queue.offer(next);
                }
            }
        }
        return count == numCourses;
    }
}
```

## 实现Trie（前缀树）
**思路：**
这道题要求实现Trie（前缀树）数据结构，支持插入、查找和前缀匹配操作。

Trie是一种树形数据结构，每个节点有26个子节点（对应26个小写字母），用布尔值isEnd标记是否是单词结尾。

**步骤：**
1. 定义TrieNode类：
   - children数组（26个TrieNode）
   - isEnd布尔值
2. insert操作：
   - 从根节点开始，遍历单词的每个字符
   - 如果对应子节点不存在，创建新节点
   - 移动到子节点
   - 最后一个字符的节点，isEnd = true
3. search操作：
   - 从根节点开始，遍历单词的每个字符
   - 如果对应子节点不存在，返回false
   - 移动到子节点
   - 返回最后一个节点的isEnd值
4. startsWith操作：
   - 类似search，但不需要检查isEnd
   - 如果所有字符都存在，返回true

```java
class Trie {
    class TrieNode {
        TrieNode[] children = new TrieNode[26];
        boolean isEnd = false;
    }
    
    private TrieNode root;
    
    public Trie() {
        root = new TrieNode();
    }
    
    public void insert(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            int index = c - 'a';
            if (node.children[index] == null) {
                node.children[index] = new TrieNode();
            }
            node = node.children[index];
        }
        node.isEnd = true;
    }
    
    public boolean search(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            int index = c - 'a';
            if (node.children[index] == null) {
                return false;
            }
            node = node.children[index];
        }
        return node.isEnd;
    }
    
    public boolean startsWith(String prefix) {
        TrieNode node = root;
        for (char c : prefix.toCharArray()) {
            int index = c - 'a';
            if (node.children[index] == null) {
                return false;
            }
            node = node.children[index];
        }
        return true;
    }
}
```

## 全排列
**思路：**
这道题要求返回数组的所有全排列。

使用回溯算法：每次选择一个未使用的数字加入当前路径，递归处理剩余数字，然后撤销选择（回溯）。当路径长度等于数组长度时，说明找到了一个排列。

**步骤：**
1. 创建结果列表和路径列表，创建used数组标记已使用的数字
2. 调用backtrack函数
3. backtrack函数：
   - 如果路径长度等于数组长度，将路径加入结果，返回
   - 遍历所有数字：
     - 如果数字已使用，跳过
     - 选择：将数字加入路径，标记为已使用
     - 递归：backtrack(剩余数字)
     - 撤销选择：从路径移除数字，取消标记

```java
class Solution {
    public List<List<Integer>> permute(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        backtrack(nums, new ArrayList<>(), new boolean[nums.length], result);
        return result;
    }
    
    private void backtrack(int[] nums, List<Integer> path, 
                          boolean[] used, List<List<Integer>> result) {
        if (path.size() == nums.length) {
            result.add(new ArrayList<>(path));
            return;
        }
        for (int i = 0; i < nums.length; i++) {
            if (used[i]) continue;
            // 选择
            path.add(nums[i]);
            used[i] = true;
            // 递归
            backtrack(nums, path, used, result);
            // 撤销选择
            path.remove(path.size() - 1);
            used[i] = false;
        }
    }
}
```

```java
class Solution {
```

使用回溯算法：记录左括号和右括号的数量。可以添加左括号的条件是left < n；可以添加右括号的条件是right < left（保证有效）。

**步骤：**

     - 添加右括号，递归，撤销

```java
class Solution {

        backtrack(n, 0, 0, new StringBuilder(), result);
        return result;

                          StringBuilder path, List<String> result) {
        if (path.length() == 2 * n) {

        }
    }
}
```

## 单词搜索
**思路：**
这道题要求判断单词是否存在于二维网格中。

使用回溯+DFS：从每个位置开始，尝试匹配单词的每个字符。使用DFS在四个方向上搜索，用临时标记避免重复访问。

**步骤：**
1. 遍历网格的每个位置，作为起点
2. 对每个起点，调用DFS函数
3. DFS函数：
   - 如果index == word.length，返回true
   - 如果位置越界或字符不匹配，返回false
   - 标记当前位置为已访问（临时修改值）
   - 在四个方向上递归搜索
   - 恢复原值（回溯）
   - 返回搜索结果

```java
class Solution {
    public boolean exist(char[][] board, String word) {
        int m = board.length;
        int n = board[0].length;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (dfs(board, word, i, j, 0)) {
                    return true;
                }
            }
        }
        return false;
    }
    
    private boolean dfs(char[][] board, String word, int i, int j, int index) {
        if (index == word.length()) return true;
        if (i < 0 || i >= board.length || j < 0 || j >= board[0].length 
            || board[i][j] != word.charAt(index)) {
            return false;
        }
        // 标记为已访问
        char temp = board[i][j];
        board[i][j] = '#';
        // 四个方向搜索
        boolean found = dfs(board, word, i + 1, j, index + 1)
            || dfs(board, word, i - 1, j, index + 1)
            || dfs(board, word, i, j + 1, index + 1)
            || dfs(board, word, i, j - 1, index + 1);
        // 恢复原值
        board[i][j] = temp;
        return found;
    }
}
```

## 分割回文串
**思路：**
这道题要求将字符串分割成所有可能的回文子串组合。

使用回溯+动态规划：先用DP预处理判断哪些子串是回文，然后回溯分割。DP[i][j]表示s[i:j]是否是回文。

**步骤：**

2. 调用backtrack函数
3. backtrack函数：
   - 如果start == s.length，将路径加入结果，返回
   - 从start到末尾，尝试每个结束位置end：
     - 如果s[start:end]是回文：
       - 选择：将子串加入路径
       - 递归：backtrack(s, end+1, path, result)
       - 撤销选择：移除子串

```java
class Solution {
    public List<List<String>> partition(String s) {
        List<List<String>> result = new ArrayList<>();
        int n = s.length();
        // dp[i][j]表示s[i:j]是否是回文
        boolean[][] dp = new boolean[n][n];
        for (int i = n - 1; i >= 0; i--) {
            for (int j = i; j < n; j++) {
                if (s.charAt(i) == s.charAt(j) && (j - i <= 2 || dp[i + 1][j - 1])) {
                    dp[i][j] = true;
                }
            }
        }
        backtrack(s, 0, new ArrayList<>(), result, dp);
        return result;
    }
    
    private void backtrack(String s, int start, List<String> path, 
                          List<List<String>> result, boolean[][] dp) {
        if (start == s.length()) {
            result.add(new ArrayList<>(path));
            return;
        }
        for (int end = start; end < s.length(); end++) {
            // 如果是回文，加入路径
            if (dp[start][end]) {
                path.add(s.substring(start, end + 1));
                backtrack(s, end + 1, path, result, dp);
                path.remove(path.size() - 1);
            }
        }
    }
}
```

## 数组中第K个最大元素
**思路：** 快速选择。类似快排，但只处理包含第k大元素的那一半。

```java
class Solution {
    public int findKthLargest(int[] nums, int k) {
        return quickSelect(nums, 0, nums.length - 1, nums.length - k);
    }
    
    private int quickSelect(int[] nums, int left, int right, int k) {
        if (left == right) return nums[left];
        int pivotIndex = partition(nums, left, right);
        if (pivotIndex == k) {
            return nums[k];
        } else if (pivotIndex < k) {
            return quickSelect(nums, pivotIndex + 1, right, k);
        } else {
            return quickSelect(nums, left, pivotIndex - 1, k);
        }
    }
    
    private int partition(int[] nums, int left, int right) {
        int pivot = nums[right];
        int i = left;
        for (int j = left; j < right; j++) {
            if (nums[j] <= pivot) {
                swap(nums, i, j);
                i++;
            }
        }
        swap(nums, i, right);
        return i;
    }
    
    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```

**思路：** 动态规划。dp[i][j]表示到达(i,j)的路径数。

```java
class Solution {
    public int uniquePaths(int m, int n) {
        // dp[i][j]表示到达(i,j)的路径数
        int[][] dp = new int[m][n];
        // 第一行和第一列都是1
        for (int i = 0; i < m; i++) {
            dp[i][0] = 1;
        }
        for (int j = 0; j < n; j++) {
            dp[0][j] = 1;
        }
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                // 可以从上方或左方到达
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }
        return dp[m - 1][n - 1];
    }
}
```

## 最小路径和
**思路：** 动态规划。dp[i][j]表示到达(i,j)的最小路径和。

```java
class Solution {
    public int minPathSum(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;
        // dp[i][j]表示到达(i,j)的最小路径和
        int[][] dp = new int[m][n];
        dp[0][0] = grid[0][0];
        // 初始化第一行和第一列
        for (int i = 1; i < m; i++) {
            dp[i][0] = dp[i - 1][0] + grid[i][0];
        }
        for (int j = 1; j < n; j++) {
            dp[0][j] = dp[0][j - 1] + grid[0][j];
        }
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                // 取上方和左方的最小值
                dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];
            }
        }
        return dp[m - 1][n - 1];
    }
}
```

## 最长回文子串
**思路：** 动态规划或中心扩展。dp[i][j]表示s[i:j]是否是回文。

```java
class Solution {
    public String longestPalindrome(String s) {
        int n = s.length();
        boolean[][] dp = new boolean[n][n];
        int start = 0;
        int maxLen = 1;
        // 单个字符都是回文
        for (int i = 0; i < n; i++) {
            dp[i][i] = true;
        }
        // 两个字符的情况
        for (int i = 0; i < n - 1; i++) {
            if (s.charAt(i) == s.charAt(i + 1)) {
                dp[i][i + 1] = true;
                start = i;
                maxLen = 2;
            }
        }
        // 长度大于2的子串
        for (int len = 3; len <= n; len++) {
            for (int i = 0; i <= n - len; i++) {
                int j = i + len - 1;
                // 如果两端字符相等且中间是回文
                if (s.charAt(i) == s.charAt(j) && dp[i + 1][j - 1]) {
                    dp[i][j] = true;
                    start = i;
                    maxLen = len;
                }
            }
        }
        return s.substring(start, start + maxLen);
    }
}
```

## 最长公共子序列
**思路：** 动态规划。dp[i][j]表示text1[0:i]和text2[0:j]的最长公共子序列长度。

```java
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int m = text1.length();
        int n = text2.length();
        // dp[i][j]表示text1[0:i]和text2[0:j]的最长公共子序列长度
        int[][] dp = new int[m + 1][n + 1];
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    // 字符匹配，长度+1
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    // 字符不匹配，取最大值
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }
        return dp[m][n];
    }
}
```

## 编辑距离
**思路：** 动态规划。dp[i][j]表示word1[0:i]转换为word2[0:j]的最少操作数。

```java
class Solution {
    public int minDistance(String word1, String word2) {
        int m = word1.length();
        int n = word2.length();
        // dp[i][j]表示word1[0:i]转换为word2[0:j]的最少操作数
        int[][] dp = new int[m + 1][n + 1];
        // 初始化：空字符串转换
        for (int i = 0; i <= m; i++) {
            dp[i][0] = i;
        }
        for (int j = 0; j <= n; j++) {
            dp[0][j] = j;
        }
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    // 字符相同，不需要操作
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    // 字符不同，取三种操作的最小值：插入、删除、替换
                    dp[i][j] = 1 + Math.min(Math.min(dp[i][j - 1], dp[i - 1][j]), dp[i - 1][j - 1]);
                }
            }
        }
        return dp[m][n];
    }
}
```

## 多数元素
**思路：** 投票算法。遇到相同元素票数+1，不同元素票数-1，票数为0时更换候选。

```java
class Solution {
    public int majorityElement(int[] nums) {
        int candidate = nums[0];
        int count = 1;
        for (int i = 1; i < nums.length; i++) {
            if (count == 0) {
                // 更换候选
                candidate = nums[i];
                count = 1;
            } else if (nums[i] == candidate) {
                count++;
            } else {
                count--;
            }
        }
        return candidate;
    }
}
```

## 颜色分类
**思路：** 三指针。用三个指针分别指向0的右边界、当前遍历位置、2的左边界。

```java
class Solution {
    public void sortColors(int[] nums) {
        int left = 0;      // 0的右边界
        int right = nums.length - 1;  // 2的左边界
        int curr = 0;      // 当前遍历位置
        while (curr <= right) {
            if (nums[curr] == 0) {
                // 交换到左边
                swap(nums, left, curr);
                left++;
                curr++;
            } else if (nums[curr] == 2) {
                // 交换到右边
                swap(nums, curr, right);
                right--;
                // 注意这里curr不增加，因为交换来的可能是0或1
            } else {
                curr++;
            }
        }
    }
    
    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```

## 下一个排列
**思路：** 从右往左找第一个降序的位置，然后从右往左找第一个大于它的数，交换后反转后面的部分。

```java
class Solution {
    public void nextPermutation(int[] nums) {
        int i = nums.length - 2;
        // 从右往左找第一个降序的位置
        while (i >= 0 && nums[i] >= nums[i + 1]) {
            i--;
        }
        if (i >= 0) {
            int j = nums.length - 1;
            // 从右往左找第一个大于nums[i]的数
            while (j >= 0 && nums[j] <= nums[i]) {
                j--;
            }
            swap(nums, i, j);
        }
        // 反转后面的部分
        reverse(nums, i + 1);
    }
    
    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
    
    private void reverse(int[] nums, int start) {
        int end = nums.length - 1;
        while (start < end) {
            swap(nums, start, end);
            start++;
            end--;
        }
    }
}
```

## 寻找重复数
**思路：** 快慢指针（类似环形链表）。将数组看作链表，用快慢指针找环的入口。

```java
class Solution {
    public int findDuplicate(int[] nums) {
        int slow = nums[0];
        int fast = nums[0];
        // 找到相遇点
        do {
            slow = nums[slow];
            fast = nums[nums[fast]];
        } while (slow != fast);
        // 找环的入口
        slow = nums[0];
        while (slow != fast) {
            slow = nums[slow];
            fast = nums[fast];
        }
        return slow;
    }
}
```
