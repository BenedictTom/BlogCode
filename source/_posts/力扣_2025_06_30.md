## 📌 今日题目：
移动零

### 📝 题目描述
给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意 ，必须在不复制数组的情况下原地对数组进行操作。

---

### 示例  

**示例 1：**  
输入: `nums = [0,1,0,3,12]`
输出: `[1,3,12,0,0]`

**示例 2：**  
输入: `nums = [0]`
输出: `[0]`

---

## 思路分析

本题的核心是**原地修改数组**并**保持非零元素的相对顺序**，最经典的解法是**双指针法**。

我们可以使用一个指针 `nonZeroIndex` 来记录下一个非零元素应该被放置的位置。

**核心思路：**
- `nonZeroIndex` 指针初始化为 0。
- 遍历整个数组（使用指针 `i`）。
- 当遇到一个非零元素 `nums[i]` 时，就将其移动到 `nonZeroIndex` 指向的位置。
- 移动后，将 `nonZeroIndex` 加 1。

这个过程有两种具体的实现方式：

**方法一：覆盖法**
1. 遍历数组，将所有非零元素按顺序覆盖到数组的前面（从 `nonZeroIndex` 开始）。
2. 完成后，所有非零元素都已按原相对顺序移动到了数组前面。
3. 此时 `nonZeroIndex` 指向第一个需要被置为 0 的位置，将数组从 `nonZeroIndex` 到末尾的所有元素都设置为 0。
- **优点**：写入操作次数最少。

**方法二：交换法**
1. 遍历数组，当 `nums[i]` 是非零元素时，将其与 `nums[nonZeroIndex]` 进行交换。
2. 交换后，`nonZeroIndex` 加 1。
- **优点**：代码更简洁，但在非零元素较多的情况下，交换操作比直接覆盖更耗时。

## ✅ Python 解法
```python
# 方法一：覆盖法
def move_zeroes_cover(nums):
    non_zero_index = 0
    # 将所有非零元素移动到数组前面
    for i in range(len(nums)):
        if nums[i] != 0:
            nums[non_zero_index] = nums[i]
            non_zero_index += 1
    
    # 将剩余部分填充为 0
    for i in range(non_zero_index, len(nums)):
        nums[i] = 0

# 方法二：交换法
def move_zeroes_swap(nums):
    non_zero_index = 0
    for i in range(len(nums)):
        if nums[i] != 0:
            # 交换非零元素到 non_zero_index 位置
            nums[i], nums[non_zero_index] = nums[non_zero_index], nums[i]
            non_zero_index += 1
```

## ✅ Java 解法

### 方法一：覆盖法
```java
class Solution {
    public void moveZeroes(int[] nums) {
        int nonZeroIndex = 0;
        // 将所有非零元素移动到数组前面
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                nums[nonZeroIndex++] = nums[i];
            }
        }
        // 将剩余部分填充为 0
        for (int i = nonZeroIndex; i < nums.length; i++) {
            nums[i] = 0;  
        }
    }
}
```

### 方法二：交换法
```java
class Solution {
    public void moveZeroes(int[] nums) {
        int nonZeroIndex = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                // 交换非零元素到 nonZeroIndex 位置
                int temp = nums[nonZeroIndex];
                nums[nonZeroIndex] = nums[i];
                nums[i] = temp;
                nonZeroIndex++;
            }
        }
    }
}
```
