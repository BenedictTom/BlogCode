## 📌 今日题目：
最长连续序列

### 📝 题目描述

给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 O(n) 的算法解决此问题。

---

### 示例  

**示例 1：**  
输入：`nums = [100,4,200,1,3,2]`  
输出：`4`  
解释：最长数字连续序列是 `[1, 2, 3, 4]`。它的长度为 4。

**示例 2：**  
输入：`nums = [0,3,7,2,5,8,4,6,0,1]`  
输出：`9`

---

## 思路分析

本题的关键在于**如何高效地找到连续序列的起始点**。

**核心思路：**
- 使用 HashSet 存储所有数字，实现 O(1) 的查找
- 对于每个数字，只有当它是连续序列的起始点时才进行计数
- 如何判断起始点：检查 `num - 1` 是否存在于集合中
  - 如果不存在，说明 `num` 是某个连续序列的起始点
  - 如果存在，说明 `num` 不是起始点，跳过

**算法步骤：**
1. 将所有数字加入 HashSet
2. 遍历每个数字 `num`
3. 如果 `num - 1` 不存在于集合中，说明 `num` 是起始点
4. 从 `num` 开始，不断检查 `num + 1` 是否存在，计算连续序列长度
5. 更新最长序列长度

**时间复杂度：** O(n)，每个数字最多被访问两次
**空间复杂度：** O(n)，用于存储 HashSet

---

## ✅ Python 解法

```python
def longest_consecutive(nums):
    if not nums:
        return 0
    
    # 将所有数字加入集合，实现 O(1) 查找
    num_set = set(nums)
    max_length = 0
    
    for num in num_set:
        # 只有当 num 是连续序列的起始点时才计算
        if num - 1 not in num_set:
            current_num = num
            current_length = 1
            
            # 从当前数字开始，不断查找下一个连续数字
            while current_num + 1 in num_set:
                current_num += 1
                current_length += 1
            
            # 更新最长序列长度
            max_length = max(max_length, current_length)
    
    return max_length
```

## ✅ Java 解法

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        if (nums.length == 0) {
            return 0;
        }
        
        // 将所有数字加入 HashSet
        Set<Integer> numSet = new HashSet<>();
        for (int num : nums) {
            numSet.add(num);
        }
        
        int maxLength = 0;
        
        for (int num : numSet) {
            // 只有当 num 是连续序列的起始点时才计算
            if (!numSet.contains(num - 1)) {
                int currentNum = num;
                int currentLength = 1;
                
                // 从当前数字开始，不断查找下一个连续数字
                while (numSet.contains(currentNum + 1)) {
                    currentNum++;
                    currentLength++;
                }
                
                // 更新最长序列长度
                maxLength = Math.max(maxLength, currentLength);
            }
        }
        
        return maxLength;
    }
}
```
